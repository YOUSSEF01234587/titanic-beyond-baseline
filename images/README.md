# Images

This folder contains screenshots and diagrams for the README.

## How to Add Images

1. Run the notebook on Kaggle or locally
2. Save the output visualizations as PNG files
3. Place them in this folder
4. Update the README.md with the image paths

## Recommended Screenshots

| Filename | Description |
|----------|-------------|
| `eda-overview.png` | Exploratory data analysis summary |
| `feature-importance.png` | Cross-model feature importance comparison |
| `ensemble-comparison.png` | Model performance comparison chart |
| `confusion-matrix.png` | Confusion matrix for best model |
| `learning-curves.png` | Learning curves for all models |
| `shap-summary.png` | SHAP summary plot for best model |
| `pipeline-overview.png` | Visual pipeline diagram |

## Image Guidelines

- **Format**: PNG (recommended) or JPG
- **Resolution**: 1200x800 minimum for clarity
- **File size**: Keep under 2MB for fast loading
- **Naming**: Use lowercase with hyphens (no spaces)

## Adding to README

```markdown
![Alt Text](images/filename.png)
```

Or with a link:

```markdown
[![Alt Text](images/filename.png)](images/filename.png)
```
